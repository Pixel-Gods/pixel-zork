This is a simple, single-page HTML text-adventure inspired by classic interactive fiction (Zork-like). The game runs entirely in the browser with no server required.

How to run
- Open `index.html` in a modern browser (double-click or drag into a browser window).
- Type commands such as:
  - go north / north / n
  - look
  - take <item>
  - drop <item>
  - inventory
  - use <item>
  - help
  - restart

Features
- Small map with rooms, items, and a simple puzzle.
- Minimal mobile-friendly UI.
- Local game state (not persisted beyond refresh unless you add save/load via localStorage).
